h1 is a heading tag this project must use the: p, div, h1, b, em and span tags

h2 is a smaller heading tag, there are more h# tags, descending tag order

p is a paragraph tag, it's a block element, which creates padding surrounding itself.
It also accepts attributes.

div is another block element, but only creates a new line pre and post element
b is for bolding
br creates a new line
italicizes text
span is an inline element, accepts attributes.
below is an anchor tag, 'a' and attribute 'href=' which points to a url, it also includes a hyperlink which is highlighted and clickable text

blow is an image tag, 'img' and attribute 'src=' which points to location of image file
it also has 'alt=' tag, this tag is for when the image file is invalid or missing
